# 🚀 Free Astro + React + Contentful template

This is a simple Astro template with contentful integration.

## 👍 Features

- Astro v1.x
- Node v16.x
- React lts
- Tailwind lts

## 🗂️ src structure

```bash
📦src
 ┣ 📂components
 ┃ ┣ 📂elements
 ┃ ┃ ┗ 📂exampleElement

 ┃ ┗ 📜index.astro
```

## 👌 Usage

1. Install Depedencies

```sh
yarn install
```

2. Run in dev mode

```sh
yarn dev
```

3. Or build source

```sh
yarn build
```
